Project Overview

Tri-Fleet is a logistics fleet management dashboard designed in Excel. It tracks vehicle performance, maintenance schedules, fuel usage, and delivery timelines. Built for efficient operations and fast decision-making.

Key Features:

  • Fleet performance tracking by vehicle type and location
  • Maintenance status and upcoming service alerts
  • Fuel efficiency analysis with trend charts
  • Delivery and dispatch performance metrics

Tools Used:

Microsoft Excel, PivotTables, Charts, Conditional Formatting
