Project Overview

Frontier is a high-level Excel dashboard for geospatial and operational analysis. Designed for teams navigating new markets and unfamiliar data terrain, it helps visualize impact, reach, and untapped opportunity.

Key Features:

  • Interactive map overlays for regional data
  • Performance indicators across multiple zones
  • Market gap analysis for new expansion areas
  • Dynamic filters and segmentation

Tools Used:

Microsoft Excel, Power Query, Custom Charts, Geospatial Mapping Add-ins
